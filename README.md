# CSA B149.2-25 Propane Storage and Handling Code

Canadian Standards Association B149.2-25 Propane Storage and Handling Code (8th Edition, February 2025).

## Overview

This repository contains the complete CSA B149.2-25 knowledge base used by the LARK Labs dev-tools MCP server for generating compliant propane safety and handling content.

## Standards Details

- **Standard**: CSA B149.2-25
- **Title**: Propane Storage and Handling Code
- **Edition**: 8th Edition
- **Published**: February 2025
- **Authority**: Canadian Standards Association (CSA)
- **Version**: 1.0.0
